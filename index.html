// setup

Ready in two steps.

01
Download VMlaunch.exe
Click the download button. No installer — it's a single portable executable. Works on Windows 10 and 11 (64-bit).
02
Install QEMU
VMlaunch is a frontend — QEMU does the actual virtualisation. Install it via:

winget install QEMU

Or download the installer from qemu.org. VMlaunch detects it automatically.
03
Get an ISO
Download any OS ISO — Ubuntu, Debian, Windows, FreeDOS, anything. Open VMlaunch, click Browse…, select your ISO, set RAM and CPUs, press ▶ LAUNCH VM.
⚠ Hardware acceleration note: WHPX (Windows Hypervisor Platform) is required for fast VMs on Windows. Enable it via: Settings → Apps → Optional Features → More Windows Features → Windows Hypervisor Platform. VMlaunch works without it but will be slower.
